---

## Правила

1. Запрещен импорт из модулей. Контракт поддерживается только на уровне пакетов, реализация в модулях является закрытой
    и может быть изменена в любой момент.
2. Запрещено добавлять все реализации в пакет. Он должен предоставлять только минимум для работы.
3. Все пакеты должны быть покрыты тестами.
4. Перед отправкой PR, убедитесь, что все тесты и линтеры проходят.
5. В пакете должна поддерживаться единая структура модулей - константы, исключения, утилиты, реализация.
6. Все пакеты и модули должны быть покрыты документацией.
